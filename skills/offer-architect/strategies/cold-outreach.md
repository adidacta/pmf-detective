# Cold Outreach Strategy Guide

## When to Use
- Building relationships with new people
- ICP is not in your existing network
- Want direct, personal feedback quickly
- Budget is limited

## Platforms

### LinkedIn
- Best for B2B
- Can see prospect's content for personalization
- Connection requests + DMs
- InMail for non-connections (premium)

### Email
- Best for targeted lists
- Can automate with tools
- Works for B2B and B2C
- Requires finding email addresses

### Reddit/Communities
- Great for niche ICPs
- Provide value before pitching
- Build credibility through comments
- DM engaged users

## Message Structure

### The AIDA Framework
1. **Attention** - Personalized opener (reference their content/work)
2. **Interest** - Connect to their pain point
3. **Desire** - Brief value proposition
4. **Action** - Soft CTA (question, not hard sell)

### LinkedIn DM Template

```
Hey [Name],

[Personalization - something specific about their recent post/work]

I noticed you're [observation about their situation/challenge].

I've been helping [similar people] with [specific problem] by [brief solution].

Would you be open to [soft ask - question, quick chat, resource]?

[Your name]
```

### Cold Email Template

```
Subject: [Benefit] for [Their company/role]

Hi [Name],

[Personalized opener - how you found them, what impressed you]

I'm reaching out because [observation about their challenge].

We help [ICP description] [achieve outcome] by [method].

[Brief social proof - "We recently helped X achieve Y"]

Would it make sense to [specific ask]?

Best,
[Your name]
```

## Follow-up Sequence

1. **Day 0:** Initial message
2. **Day 3:** First follow-up (add value, share insight)
3. **Day 7:** Second follow-up (different angle)
4. **Day 14:** Final follow-up (break-up email)

## Personalization Tips

1. Reference their recent content/post
2. Mention a mutual connection
3. Note something specific about their company
4. Connect their challenge to your solution
5. Keep it conversational, not salesy

## Metrics to Track

- Messages sent per day
- Response rate (aim for 10-20%)
- Positive response rate
- Meetings/calls booked
- Conversions to mini-product

## Tools

- LinkedIn Sales Navigator (finding prospects)
- Hunter.io (finding emails)
- Lemlist, Mailshake (email automation)
- Notion/Sheets (tracking responses)
